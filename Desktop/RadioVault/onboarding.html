RadioVault

Radio Broadcast Archive & Clipping Tool

Get Started

Import a team config to connect to your team's shared database, or set up manually with an RSS feed.

Add an RSS Feed

Paste the podcast or RSS feed URL for the radio broadcasts you want to archive and search.

API Keys (Optional)

For transcription, add a Groq API key (or use Local Whisper). For AI summaries, add an Anthropic key. You can add these later in Settings.

Ready to Go

You can add feeds, API keys, and team config anytime in Settings.
