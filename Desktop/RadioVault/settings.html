Settings

Team Setup

Import shared feed and Supabase settings. Transcription and AI summary API keys are personal; add your own below or use Local Whisper.

RSS Feeds

Add podcast or RSS feed URLs. RadioVault will discover episodes and download audio for transcription.

Supabase Sync

Cloud database for syncing transcripts and clips across machines.

Shared Media Path

Point this to a folder on CAMS (or any shared drive). Audio downloads will be stored here so everyone on the team can play and export without re-downloading.

Personal API Keys

Bring your own API keys for cloud features. They are stored in macOS Keychain and are never exported in team config files.

Used for Groq cloud transcription.

Used for OpenAI transcription and/or OpenAI summaries.

Used for Claude summaries and key moment detection after transcription.

Used only with the custom OpenAI-compatible transcription endpoint below.

Transcription

Use Local Whisper on-device, or bring your own API key for Groq, OpenAI, or another OpenAI-compatible transcription service. For long broadcasts (2+ hours), Local Whisper is recommended.

Default is whisper-1 because RadioVault needs segment timestamps for search and clipping.

The endpoint must accept OpenAI-style multipart audio transcription requests and return text plus segments.

Taxonomy Editor

Keyword categories used for clip extraction from broadcast transcripts.

Ingestion

Transcribe downloaded episodes and extract clips. AI summaries are generated if an Anthropic API key is configured.


  


  
  

    
Storage

    
Downloaded audio files are cached locally. Transcribed episodes can be safely cleared -- audio re-downloads on demand when you need to export a clip.
